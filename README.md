# 🔐 Password Strength Analyzer

## 📌 Project Overview

The Password Strength Analyzer is a Cyber Security web application that helps users evaluate the strength of their passwords. It checks whether a password is Weak, Medium, or Strong based on various security rules and provides suggestions to improve password security.

This project also includes a simple Login and Registration system along with a Dark Mode feature to enhance the user experience.

## ⚙️ Password Evaluation Criteria

The application checks for:

✔ Password Length (Minimum 8 characters)

✔ Uppercase Letters

✔ Lowercase Letters

✔ Numbers

✔ Special Characters

---

## 📊 Password Strength Levels

| Score | Strength |
|--------|----------|
| 0 - 2 | Weak |
| 3 - 4 | Medium |
| 5 | Strong |

## 🚀 How to Run the Project

### Step 1

Download or Clone the project.

### Step 2

Open the project folder.

### Step 3

Double-click **register.html**

### Step 4

Create a new account.

### Step 5

Open **login.html**

### Step 6

Login using your registered credentials.

### Step 7

After successful login, the Password Strength Analyzer dashboard will open.
